01- Emmet

Work Flow - Markup & Styling

01- Nesting >

Example:

ul>li>a ↔️

Result:

<ul> <li><a href=""></a></li> </ul>

02- Sibling +

Example:

section+header+nav ↔️

Result:

<section></section> <header></header> <nav></nav>

03- Repeat *

Example:

ul>li*3 ↔️

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04- Class .

Example:

div.container ↔️

Result:

<div class="container"></div>

05- Classes . . .

Example:

div.class1.class2.class3 ↔️

Result:

<div class="class1 class2 class3"></div>

06- Id #

Example:

nav#menu ↔️

Result:

<nav id="menu"></nav>

07- Text {}

Example:

p{Lorem Ipsum Dolor} ↔️

Result:

<p>Lorem Ipsum Dolor</p>

08- Attributes []

Example:

a[href=www.google.com] ↔️

Result:

<a href="www.google.com"></a>

09- Numbering $

Example:

ul>li.num$*3 ↔️

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10- Reverse @-

Example:

ul>li.num$@-*3 ↔️

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11- Base @n

Example:

ul>li.num$@10*3 ↔️

Result:

<ul> <li class="num10"></li> <li class="num11"></li> <li class="num12"></li> </ul>

12- Climp Up ^

Example:

section>article>div^p ↔️

Result:

<section> <article> <div></div> </article> <p></p> </section>

13- Grouping ()

Example:

(header>nav)+(section>article) ↔️

Force trigger: ctrl + space

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14- Lorem

Example:

lorem5 ↔️

Result:

Lorem ipsum dolor sit amet.

15- Document

Example:

! ↔️

Result:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> </html>

16- Meta

Example:

meta:vp ↔️

Result:

<meta name="viewport">

17- Callenge

Example:

header#header>nav.navbar.navbar-inverse.navbar-fixed-top>div.container>(div.navbar-header>button.navbar-toggle[type=button data-toggle=collapse data-target=.navbar-collapse]>span.sr-only{Toggle navigation}+span.icon-bar*3^a.navbar-brand[href=index.php]>img[src=imgs/logo.png])+(div.collapse.navbar-collapse.navbar-right>ul.nav.navbar-nav>(li>a.link[href=index.html]{Inicio})+(li>a.active.link[href=login.html]{Ingresar})+(li>a.link[href=registro.html]{Registrarse})+(li>a.link[href=contacto.html]{Contacto})) ↔️
